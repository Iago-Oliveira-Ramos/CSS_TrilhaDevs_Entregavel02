PROJETO 01

SITE PARA CINEMA

Descrição: Site de um cinema que permite ao usuário ver os filmes disponíveis e deletar ou adicionar o quanto deles desejar.
Funcionalidades: O usuário pode digitar o nome do filme que deseja adicionar na caixa de texto e clicar no botão verde "Add" para realizar a ação. O filme irá aparecer no final da lista. Ele também pode deleter qualquer um clicando no botão vermelho "delete".
Linguagens utilizadas: HTML5, CSS3 e Javascript
Repositório no GitHub: https://github.com/Iago-Oliveira-Ramos/Projeto01

PROJETO 02

JOGO DA VELHA

Descrição: Site para poder jogar jogo da velha com outra pessoa ou contra uma I.A.
Funcionalidades: No menu inicial, o usuário pode selecionar se deseja jogar com outra pessoa ou uma I.A.. Dentro do jogo ele só precisa clicar dentro dos quadrados para adicionar os "Xs" e "Os". As velhas e vitórias são anunciadas pelo sistema e o placar é atualizado automaticamente.
Linguagens utilizadas: HTML5, CSS3 e Javascript
Repositório no GitHub: https://github.com/Iago-Oliveira-Ramos/Projeto02
